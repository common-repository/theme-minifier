== Description ==

Minifies any theme with one click, reducing bandwidth and lowering page loading times.

* Simple "one-click" interface
* Creates copy of theme, leaving original intact
* Reduces bandwidth
* Lowers page loading times
* Compresses images
* Minifies HTML markup, CSS, and Javascript

== Installation ==

1. Upload the 'theme-minifier' folder to your plugins folder: 'wp-content/plugins/'
1. Activate 'Theme Minifier' in the 'Plugins' menu in WordPress
1. Use the plugin (located under Settings->Theme Minifier)
